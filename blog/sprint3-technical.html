What Functions Are, And Why Theyre Useful

Functions are one of, if not the most, useful tools in coding. Functions are essentially bits of code that can be used over and over by being called. Functions have ‘parameters’, which are placeholder text that is then entered as arguments when the function is called. This is shown in the example below, where the function ‘addNumbers’ adds two numbers together, which are passed into the function when it is called.

While this is a super simple example, functions can do really complicated things, and allow much more condensed code, as you can repeat the block of code over and over by simply calling the function rather than re-writing it every time.
